O **painel principal (Página Inicial)** do **VendaAI** é uma interface **intuitiva e organizada** para a **gestão de vendas, produtos e clientes**.

## Criando uma Página Inicial

Para **criar um conteúdo na Página Inicial**, siga os passos abaixo:

<ListaCard>

1. Clique no botão **"Configurações"**.
2. Selecione a opção **"Criar Conteúdo"**, conforme ilustrado na **imagem a seguir**:

</ListaCard>

![Acessando o VendaAI](https://i.imgur.com/YvXWuMQ.png)

Após iniciar a criação, preencha os **campos disponíveis** para inserir o **conteúdo desejado**, conforme descrito nos **tópicos a seguir**:

![Acessando o VendaAI](https://i.imgur.com/KVgC4pz.png)

Preencha os seguintes **campos** para configurar o conteúdo:

<ListaCard>

1. **Empresa** – Define a **empresa** na qual o conteúdo será exibido.
2. **Filial** – Define a **filial** onde o conteúdo aparecerá.
3. **Nome** – Descrição atribuída ao **novo conteúdo**.
4. **Perfil** – Permite escolher se o conteúdo será exibido para **clientes**, **vendedores** ou **ambos**.
5. **Conteúdo** – Campo destinado à inclusão de **imagens, vídeos e textos**, que serão exibidos na **tela inicial**.
6. **Está Ativo** – Define se o conteúdo criado estará **ativo ou inativo**.

</ListaCard>

![Configurações de datas e código](https://i.imgur.com/s4a4bKG.png)

E os campos adicionais de período e identificação:

<ListaCard>

1. **Data inicial** – Define a **data de início da exibição** do conteúdo.
2. **Data final** – Define a **data de término da exibição** do conteúdo.
3. **Código** – Campo de **preenchimento livre** para identificação do conteúdo.

</ListaCard>

Após preencher os **campos necessários**, clique em **“Salvar”** para disponibilizar o conteúdo na **Página Inicial**.
