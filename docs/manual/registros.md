## Empresas

Como exemplo da seção de registros, a área de **Empresas** permite consultar todas as empresas cadastradas no sistema ou registrar novas empresas.

Para realizar o cadastro, é necessário informar:

<ListaCard>

- **Nome**
- **Código**
- **Status** (ativo ou inativo)

</ListaCard>

Após o cadastro, é possível pesquisar as empresas registradas, visualizando:

<ListaCard>

- **Nome**
- **Código**
- **Status**

</ListaCard>

Além disso, a interface disponibiliza opções para **editar** ou **excluir** registros existentes, facilitando a manutenção das informações.

![Acessando o VendaAI](https://i.imgur.com/2Pp2z6Z.png)

A pesquisa é intuitiva e facilita a localização das empresas cadastradas, proporcionando uma gestão mais organizada e ágil das informações.

Para **registrar uma nova empresa**, clique no botão **“+ Criar nova empresa”**, localizado no canto superior direito da tela, conforme ilustrado na imagem abaixo:

![Acessando o VendaAI](https://i.imgur.com/U4cmgPe.png)

Após clicar no botão de criação, será exibido o pop-up **Criar Empresa**.

Nele, é necessário preencher os seguintes campos:

<ListaCard>

- **Nome**
- **Código**
- **Está ativo?**

</ListaCard>

Após o preenchimento das informações, confirme o cadastro para concluir a criação da empresa.

Segue abaixo a imagem de instrução:

![Acessando o VendaAI](https://i.imgur.com/OwRUBeQ.png)

<ListaCard>

1. Insira o **nome da empresa**.
2. Informe o **código da empresa**.
3. Caso deseje que a empresa fique ativa no **VendaAI**, marque a opção **"Está ativo?"**. Caso contrário, deixe a opção desmarcada.
4. Após preencher as informações, clique em **Salvar**.

</ListaCard>

Pronto! A empresa será criada no **VendaAI**.
